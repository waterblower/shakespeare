I always liked Shakespeare when I was young. It was quite a journey for me since I tried to learn English through Shakespeare (not the best option for learning English as a second language). I have enjoyed many of BBC's Shakespeare productions when I was in high school.

Recently I started to work intensely in AI video productions and was having fun with Hermes agent. Yesterday, this idea popped up in my mind: Why not generating a complete set of reference images for plays of Shakespeare? So that everyone can direct their own Shakespeare dramas.

Therefore, I asked Hermes to download All's Well That Ends Well from https://shakespeare.mit.edu/ ([only 64 stars on GitHub?](https://github.com/TheMITTech/shakespeare/tree/master)) and here we go.

The image production pipeline is done by [Omni Artist](https://github.com/waterblower/Omni-Art-Skills), a collection of Hermes skills which I wrote. Feel free to use these assets to produce your own Shakespeare dramas.
