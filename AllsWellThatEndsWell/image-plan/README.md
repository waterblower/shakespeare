# All's Well That Ends Well image-plan

This directory contains English TOML image generation plans derived from `../shot-groups/`.

There are 5 TOML files because the play has 5 acts. Execute them in act order: `1.toml` -> `2.toml` -> `3.toml` -> `4.toml` -> `5.toml`.

Inside each act file, the task order is:

1. environments
2. tools / props / clothing references
3. character identity assets
4. story concept stills

The TOML files only define generation tasks; no images have been generated. Target assets are written under `../assets/`.
